# GitHub Repository Setup

## Anleitung zum Pushen
1. Initialisiere Git: `git init`
2. Remote hinzufügen: `git remote add origin https://github.com/stefanschulz/clubsuite-training.git`
3. Dateien hinzufügen: `git add .`
4. Commit: `git commit -m "Initial release 0.1.0"`
5. Push: `git push -u origin main`
