# Support

## Submitting Bugs

If you encounter a bug, please create an issue on GitHub.
**Please Include:**
- App Version (see `appinfo/info.xml`)
- Nextcloud Version
- PHP Version
- Browser / OS (if frontend related)
- Steps to reproduce
- Relevant logs (Anonymized!)

## Privacy & GDPR

**Important:** Do not include real personal data (XML exports, names, IBANs) in public issues.
This app supports the Nextcloud Privacy API for data export and deletion.
